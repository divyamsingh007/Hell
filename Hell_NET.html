Installation

Install HellNET-UI's source files via npm. We take care of injecting the Carbon Encryptions needed.

$ npm install @HellNET-ui/core
or use a CDN (not recommeded).

Load the following : X-UA-Compatible

  1.  Kali Linux / Garuda Linux
  2.  Strong VPN Connection (optional but recommeded)
  3.  Metasploit Framework

* Many of our components require the use of JavaScript to function. Specifically, they require our own JavaScript plugins.

  • .zip
  • .liveNode
  • NetInstaller
  • Lite (for Low-End PC)
